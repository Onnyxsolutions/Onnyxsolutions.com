# Onnyx Solutions Website

This is a simple static website for Onnyx Solutions, the customer-facing brand of Onnyx Inc.

## Files
- `index.html`
- `style.css`

## Updates included
- Smooth scrolling navigation
- Background contact form submissions through FormSubmit
- Required form fields
- Better input focus styling

## Important
The contact form currently sends in the background to:
`info@onnyxsolutions.com`

## First-time setup
FormSubmit may send one activation email to `info@onnyxsolutions.com` the first time the form is submitted.
Open that email and confirm the address so future submissions arrive normally.
